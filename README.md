# SpaceAI – Interactive AI Learning Platform

## Features

* User Login System
* Interactive Planet Learning
* AI Space Tutor
* Space Quiz Module
* Planet Comparison
* Progress Tracking
* Certificate Generation

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## Project Overview

SpaceAI is an interactive educational platform designed to help students learn about the Solar System through AI-powered tutoring, quizzes, planet exploration, and certificate-based learning.
